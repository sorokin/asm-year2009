Запуск KolibriOS под эмулятором bochs

Прежде чем начинать делать второе задание, необходимо научится пользоваться эмулятором bochs. Ниже приводится последовательность действий необходимая для того, чтобы запустить KolibriOS под эмулятором bochs.

Вначале необходимо скачать с сайта kolibrios.org архив с образом дискеты. Файл образ называется kolibri.img и имеет размер 1.47Мб. Распакуйте его и положите в текущий каталог.

bochs это программа, которая эмулирует компьютер с x86-совместимым процессором. Информация о том какой именно компьютер эмулируется хранится в конфигурационном файле. Поэтому для начала нам необходимо создать этот конфигурационный файл. Запустим bochs:

$ bochs

Далее выбираем:

3. Edit options
11. Disk & Boot options
1. First Floppy Drive
What type of floppy drive? [none] 3.5" 1.44M
Enter new filename, or 'none' for no disk: [none] kolibri.img
What type of floppy media? (auto=detect) [none] 1.44M
Is media write protected? [no] no
Is media inserted in drive? [no] yes
0. Return to previous menu
0. Return to previous menu
4. Save options to...
Save configuration to what file? To cancel, type 'none'. [none] kolibri.bxrc
7. Quit now

В любом вопросе вы можете ввести "?", чтобы узнать список возможных вариантов. После выхода в текущем каталоге появится файл kolibri.bxrc. Запустим его используя команду:

$ bochs -f kolibri.bxrc

Нажав на кнопку с мышкой на панели инструментов можно подвигать мышкой «внутри» bochs. Чтобы выйти из этого режима нужно нажать Ctrl+средняя кнопка мыши.
